# 简牍 (Jian Du) 快速使用指南

## 🚀 开始使用

### 1. 准备工作
1. 用浏览器打开 `index.html`
2. 等待引擎启动（左侧显示"引擎就绪"）
3. 输入你的 DeepSeek API Key（本地安全存储，不会上传）

### 2. 上传文件
- 点击 **"+"** 按钮添加文件
- 支持格式：`.xlsx` (Excel) 和 `.csv`
- 可上传多个文件
- 上传后可见文件预览（行数×列数 + 前几列名称）

### 3. 移除文件
- 单个移除：点击文件右侧的 **×** 按钮
- 全部清除：点击红色 **×** 按钮

### 4. 输入分析需求
在文本框中用自然语言描述你的需求，例如：

**简单分析:**
```
统计每个产品的总销售额，按降序排列
```

**数据比对:**
```
比对来源 A 和 B，找出 ID 相同但单价不符的记录
```

**数据清洗:**
```
删除重复记录，填补缺失值，筛选出销售额大于 10000 的订单
```

**图表生成:**
```
绘制月度销售趋势图，用折线图展示
```

### 5. 执行分析
点击 **"开启自动化研读"** 按钮，等待 AI 处理完成。

### 6. 查看结果
- 分析简报：AI 生成的文字说明
- 图表：如果有生成图表，会显示在下方
- 下载报告：点击 **"📥 下载《简牍分析报告》"** 按钮

## 💡 高级技巧

### 混合文件处理
可以同时上传 Excel 和 CSV 文件进行联合分析：
```
合并 sales_2024.xlsx 和 customer_info.csv，按客户 ID 关联，计算每个客户的总消费
```

### 多步骤分析
复杂任务可以分步骤进行：
1. 第一步："筛选出销售额大于 10000 的订单"
2. 下载结果后，重新上传
3. 第二步："按产品分类统计"

### 处理大文件
- 超过 10 万行的文件建议分批处理
- 可以先让 AI 生成抽样分析代码
- 使用 `df.sample(n=1000)` 进行快速探索

## ⚠️ 常见问题

### Q: 上传 CSV 后报错 "Excel file format cannot be determined"
**A:** 这是旧版本的 bug，已修复。如果仍遇到：
1. 确认文件确实是 CSV 格式（可用记事本打开查看）
2. 检查文件扩展名是否为 `.csv`
3. 尝试用 Excel 另存为 UTF-8 编码的 CSV

### Q: 引擎启动很慢
**A:** 首次启动需要加载 Pyodide 核心（约 300MB）：
- 等待 45 秒内完成
- 如果超时，刷新页面重试
- 确保网络连接正常

### Q: AI 生成的代码报错
**A:** 系统会自动尝试修复（最多 3 次）：
1. 查看错误日志，了解具体问题
2. 简化你的分析需求
3. 将复杂任务拆分成多个简单任务

### Q: 图表无法显示
**A:** 检查以下几点：
1. 确保 AI 代码中调用了 `embed_chart()`
2. 图表文件名必须是 `c1.png`, `c2.png` 等
3. 汉字引擎需要加载中文字体（约 5MB）

### Q: 内存不足
**A:** 浏览器内存有限：
1. 移除不需要的文件
2. 刷新页面清空内存
3. 大文件分批处理

## 🔒 安全说明

- **数据本地处理**: 所有文件在浏览器内处理，不会上传到服务器
- **API Key 安全**: Key 仅用于调用 DeepSeek API，不会保存或泄露
- **隐私保护**: 处理完成后，刷新页面即可清除所有数据
