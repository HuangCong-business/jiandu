# 🔧 快速修复说明

## 问题 1: API Key 无效 (401)

### 原因
DeepSeek API Key 未填写或已失效

### 解决方法
1. **获取新 Key:**
   - 访问 https://platform.deepseek.com
   - 注册/登录账号
   - 进入"API Keys"页面
   - 创建新密钥并复制

2. **填写 Key:**
   - 打开首页 `index.html`
   - 点击"开始新分析"
   - 在左侧"密钥配置"输入框粘贴 Key
   - 系统会自动保存

## 🎯 正确的使用流程

### 第一次使用
1. **双击** `启动服务器.bat` 启动 HTTP 服务器
2. **打开浏览器** 访问 http://localhost:8080
3. **看到首页**（紫色渐变背景，4 个卡片）
4. **点击"开始新分析"** 进入工作界面
5. **填写 API Key**（自动保存）
6. **上传文件**（显示预览表格）
7. **输入分析需求** → 执行

### 以后使用
1. **双击** `启动服务器.bat`（或开机自动启动）
2. **访问** http://localhost:8080
3. **首页会自动显示历史记录**
4. **点击记录直接加载** 或 开始新分析

## 🆘 如果还有问题

### 控制台错误
按 F12 打开开发者工具，查看 Console 标签的错误信息

### 页面空白
1. 检查 HTTP 服务器是否启动
2. 检查浏览器地址是否正确（http://localhost:8080）
3. 清除浏览器缓存后重试

### API 错误
1. 确认 API Key 正确（无空格）
2. 检查 DeepSeek 账户余额
3. 访问 https://platform.deepseek.com 验证 Key 有效性
