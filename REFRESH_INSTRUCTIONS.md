# 🔄 强制刷新说明

## 为什么需要强制刷新？

浏览器可能缓存了旧版本的 `app.js` 文件，导致修复代码没有生效。

## 如何强制刷新

### Windows / Linux
按 **Ctrl + Shift + R** 或 **Ctrl + F5**

### macOS
按 **Cmd + Shift + R**

### 或者手动清除缓存
1. 按 F12 打开开发者工具
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"

## 验证刷新成功

1. 打开浏览器开发者工具（F12）
2. 切换到 **Console** 标签
3. 刷新页面
4. 应该看到引擎启动进度：
   - "加载 Pyodide 核心..."
   - "安装数据科学包..."
   - "配置 Excel 支持..."
   - "引擎就绪"

## 测试 CSV 读取

1. 上传 `test_data.csv` 文件
2. 输入："统计总销售额"
3. 点击执行
4. 查看控制台输出

**成功标志：**
- ✅ 看到 "=== AI 原始代码 ===" 和 "=== 预处理后代码 ===" 日志
- ✅ 可能看到 "⚠️ 自动纠正：检测到用 read_excel 读取 CSV 文件" 
- ✅ 不再出现 "Excel file format cannot be determined" 错误
- ✅ 正常显示分析结果

## 如果还是报错

1. 展开错误面板中的 "📝 查看 AI 生成的代码"
2. 截图并发送给技术支持
3. 或者尝试用 Excel 打开 CSV 文件，确认文件格式正确
