# 📝 书法图片使用说明

## 需要准备的图片

**文件名:** `言出法随.png`

**尺寸建议:** 
- 首页：180x180 像素
- 工作页：160x160 像素

**颜色:** 白色（或任意颜色，代码会自动处理）

**位置:** 放在 `D:\简牍_V2.0\` 根目录下

---

## 🎨 图片效果

代码会自动应用以下滤镜：

```css
filter: brightness(0) invert(1);  /* 转成白色 */
opacity: 0.15;                     /* 15% 透明度 */
```

**无论你放什么颜色的图片，都会自动变成白色半透明效果！**

---

## 🖼️ 如何获取图片

### 方案 1: 使用 AI 生成（推荐）

用你提供的书法字图片：

1. 打开图片编辑软件（Photoshop、美图秀秀等）
2. 裁剪成正方形
3. 保存为 `言出法随.png`
4. 放到 `D:\简牍_V2.0\` 目录

### 方案 2: 在线生成

访问书法生成网站：
- https://aqicn.org/mobi/chinese-calligraphy/
- https://www.guoshi.com/shufa/

输入"言出法随"，生成后下载 PNG

### 方案 3: 暂时不用图片

如果暂时没有图片，系统会显示一个占位文字（需要修改代码）

## 🔧 调整效果

### 如果图片太亮/太暗

修改 `opacity` 值：

**首页 (index.html):**
```css
.calligraphy-img {
    opacity: 0.15;  /* 改这个值：0.1-0.3 之间 */
}
```

**工作页 (workspace.html):**
```css
.calligraphy-img {
    opacity: 0.12;  /* 改这个值 */
}
```

### 如果图片太大/太小

修改 `width` 和 `height`:

```css
.calligraphy-img {
    width: 180px;   /* 改这个 */
    height: 180px;  /* 改这个 */
}
```
