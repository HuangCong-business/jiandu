## 📋 下一步操作

### 1️⃣ 在 GitHub 创建仓库

1. **访问:** https://github.com/new
2. **仓库名:** `jiandu` (或其他你喜欢的名字)
3. **可见性:** 公开 (Public) 或 私有 (Private)
4. **不要** 初始化 README、.gitignore 或 license
5. 点击 "Create repository"

### 2️⃣ 推送代码到 GitHub

复制以下命令（替换为你的 GitHub 用户名）：

```bash
cd D:\简牍_V2.0
git remote add origin https://github.com/你的用户名/jiandu.git
git branch -M main
git push -u origin main
```

### 3️⃣ 部署到 Vercel

1. **访问:** https://vercel.com/new
2. **登录:** 使用 GitHub 账号登录
3. **导入项目:** 选择 `jiandu` 仓库
4. **配置:**
   - Framework Preset: `Other`
   - Build Command: 留空
   - Output Directory: 留空
5. 点击 "Deploy"
6. 等待 1-2 分钟完成

### 4️⃣ 访问部署后的网站

部署完成后，你会获得链接：
```
https://jiandu-xxx.vercel.app
```

任何人都可以访问这个链接使用简牍！

## 🎯 部署后优化

### Pyodide CDN 加速（可选）

如果加载速度慢，可以修改 `workspace.html`:

```javascript
// 从 CDN 加载 Pyodide (更快)
pyodide = await loadPyodide({ 
    indexURL: 'https://cdn.jsdelivr.net/pyodide/v0.23.4/full/'
```

### 自定义域名（可选）

在 Vercel 设置中配置自定义域名：
1. Settings → Domains
2. 添加你的域名
3. 按提示配置 DNS
