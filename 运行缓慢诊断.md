# 🐌 运行缓慢诊断指南

## ⏱️ 正常运行时间

| 阶段 | 时间 | 说明 |
|------|------|------|
| Pyodide 加载 | 30-45 秒 | 首次访问需要 |
| AI 生成代码 | 5-10 秒 | DeepSeek API 调用 |
| Python 执行 | 5-15 秒 | 数据处理 |
| 图表绘制 | 5-10 秒 | matplotlib 渲染 |
| **总计** | **45-80 秒** | 首次访问 |

---

## 🔍 如果超过 2 分钟

### 可能原因 1: AI 代码生成失败

**症状:**
- 一直显示"AI 正在深度研读..."
- 没有错误提示
- 超过 1 分钟没反应

**检查方法:**
1. 按 F12 打开控制台
2. 查看是否有错误
3. 查看 Network 标签，看 API 请求状态

**解决:**
- 检查 API Key 是否正确
- 检查网络连接
- 简化分析需求

---

### 可能原因 2: 大文件处理慢

**症状:**
- Python 执行阶段很慢
- 文件超过 10 万行

**检查:**
```
文件行数：?
文件大小：? MB
```

**解决:**
- 抽样分析（前 1000 行）
- 分批次处理
- 简化分析逻辑

---

### 可能原因 3: 图表绘制卡住

**症状:**
- 代码执行完成
- 图表一直不显示
- 控制台有错误

**检查:**
```javascript
// 控制台查看
console.log("图表生成中...")
```

**解决:**
- 刷新页面重试
- 简化图表类型
- 检查字体文件

---

### 可能原因 4: 死循环/错误重试

**症状:**
- 反复尝试执行
- 控制台显示多次"=== AI 原始代码 ==="

**检查:**
```
MAX_FIX = 3  // 最多重试 3 次
```

**解决:**
- 查看错误日志
- 简化分析需求
- 手动修正 AI 代码

---

## 🎯 箱线图测试优化

### 推荐分析需求

**✅ 好的需求（快速）:**
```
绘制收盘价的箱线图
```

**❌ 复杂需求（慢速）:**
```
分析所有列的统计数据，绘制多个图表，包括箱线图、折线图、直方图...
```

### 优化建议

1. **单次一个图表**
   ```
   绘制收盘价的箱线图
   ```

2. **指定数据范围**
   ```
   绘制最近 1000 个交易日收盘价的箱线图
   ```

3. **简单统计**
   ```
   计算收盘价的平均值、中位数、标准差
   ```

## 🚀 加速建议

### 1. 使用 CDN（部署后）

修改 `workspace.html`:
```javascript
pyodide = await loadPyodide({ 
    indexURL: 'https://cdn.jsdelivr.net/pyodide/v0.23.4/full/' 
});
```

**效果:** 加载时间 30 秒 → 15 秒

### 2. 浏览器缓存

**首次访问:** 45-80 秒  
**后续访问:** 10-20 秒（缓存生效）

### 3. 简化分析

**复杂分析分多次:**
```
第一次：绘制收盘价箱线图
第二次：绘制成交量直方图
第三次：计算相关系数
```
