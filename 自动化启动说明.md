# 🚀 简牍自动化启动指南

## 方案对比

| 方案 | 文件 | 优点 | 缺点 |
|------|------|------|------|
| **方案 1** | `启动服务器.bat` | 简单直接，可见状态 | 有命令行窗口 |
| **方案 2** | `启动服务器 (隐藏窗口).vbs` | 无窗口，优雅 | 看不到状态 |
| **方案 3** | `添加到开机启动.bat` | 开机自动，最方便 | 需要手动取消 |

---

## 📋 使用方法

### 方案 1: 一键启动（推荐日常使用）

**双击文件:** `启动服务器.bat`

**效果:**
- ✅ 显示启动状态
- ✅ 显示访问地址
- ✅ 按 Ctrl+C 可停止

**适用场景:** 每天手动启动

---

### 方案 2: 隐藏窗口启动

**双击文件:** `启动服务器 (隐藏窗口).vbs`

**效果:**
- ✅ 无窗口，后台运行
- ✅ 不占用任务栏

**停止方法:**
1. 打开任务管理器
2. 找到 `python.exe` 进程
3. 结束进程

**适用场景:** 不希望看到命令行窗口

---

### 方案 3: 开机自动启动（最推荐）

**步骤:**
1. **双击** `添加到开机启动.bat`
2. 等待提示"已添加到开机启动"
3. 完成！

**效果:**
- ✅ 每次开机自动启动
- ✅ 无需手动操作

**取消方法:**
1. 按 `Win + R`
2. 输入 `shell:startup`
3. 删除 `简牍服务器.lnk`

**适用场景:** 每天都用，不想每次手动启动

---

## 🛑 停止服务器

### 方法 1: 使用停止脚本
**双击:** `停止服务器.bat`

### 方法 2: 手动停止
- **方案 1:** 在命令行窗口按 `Ctrl + C`
- **方案 2:** 任务管理器结束 `python.exe`

## 🌐 访问地址

启动后，在浏览器访问：

```
http://localhost:8080
```

或

```
http://127.0.0.1:8080
```

---

## 💡 进阶技巧

### 修改端口号

如果 8080 端口被占用，可以修改：

1. 右键编辑 `启动服务器.bat`
2. 找到 `python -m http.server 8080`
3. 改成其他端口，如 `8888`
4. 保存

### 固定 IP 访问

如果需要局域网访问：

```bash
python -m http.server 8080 --bind 0.0.0.0
```

修改 `启动服务器.bat` 中的命令即可。

---

## ⚠️ 注意事项

1. **防火墙:** 首次启动可能提示防火墙，选择"允许"
2. **端口占用:** 如果 8080 被占用，改用其他端口
3. **Python 环境:** 确保已安装 Python 并添加到 PATH
4. **安全提示:** 服务器仅在本地运行，外部无法访问

---

## 🎯 推荐配置

**最佳实践:**
- 运行 `添加到开机启动.bat` 一次
- 以后开机自动启动
- 需要停止时用 `停止服务器.bat`

这样完全不需要手动操作！
