## ✅ 推荐方案 1: Vercel（最简单）

### 为什么选 Vercel
- ✅ 完全免费
- ✅ 自动 HTTPS
- ✅ 全球 CDN
- ✅ 自动部署（Git 推送即更新）
- ✅ 支持自定义域名

### 部署步骤

#### 第一步：准备代码
```bash
# 确保你的代码在 Git 仓库中
cd D:\简牍_V2.0
git init
git add .
git commit -m "Initial commit"
```

#### 第二步：推送到 GitHub
1. 在 GitHub 创建新仓库（如 `jiandu`）
2. 推送代码：
```bash
git remote add origin https://github.com/你的用户名/jiandu.git
git push -u origin main
```

#### 第三步：部署到 Vercel
1. 访问 https://vercel.com
2. 用 GitHub 账号登录
3. 点击 "New Project"
4. 选择你的 `jiandu` 仓库
5. 点击 "Deploy"
6. 等待 1-2 分钟完成

#### 第四步：访问
部署完成后，你会获得一个链接：
```
https://jiandu-xxx.vercel.app
```

### 优点
- 🎉 5 分钟搞定
- 🎉 无需配置
- 🎉 自动 HTTPS
- 🎉 全球 CDN 加速

## ✅ 推荐方案 2: GitHub Pages

### 部署步骤

#### 第一步：准备仓库
同上，先推送到 GitHub

#### 第二步：启用 Pages
1. 进入 GitHub 仓库页面
2. 点击 "Settings" → "Pages"
3. Source 选择 "main branch"
4. 点击 "Save"

#### 第三步：访问
等待 2-5 分钟，访问：
```
https://你的用户名.github.io/jiandu/
```

### 优点
- 🎉 完全免费
- 🎉 GitHub 原生支持
- 🎉 稳定可靠

### 缺点
- ⚠️ 国内访问可能慢
- ⚠️ 没有自动 HTTPS（需要配置）

## ✅ 推荐方案 3: Cloudflare Pages

### 部署步骤

#### 第一步：访问 Cloudflare
https://pages.cloudflare.com

#### 第二步：连接 GitHub
1. 用 GitHub 账号登录
2. 选择你的 `jiandu` 仓库
3. 点击 "Begin setup"

#### 第三步：部署
- Build command: 留空（静态网站不需要）
- Build output directory: `/`
- 点击 "Save and Deploy"

#### 第四步：访问
```
https://jiandu.pages.dev
```

### 优点
- 🎉 国内访问快
- 🎉 免费 HTTPS
- 🎉 支持自定义域名

## 🔧 常见问题

### Q1: pyodide 文件夹太大怎么办？

**方案 A: 使用 CDN**
修改 `workspace.html`：
```javascript
pyodide = await loadPyodide({ 
    indexURL: 'https://cdn.jsdelivr.net/pyodide/v0.23.4/full/' 
});
```

**方案 B: 保留本地**
- 本地部署：保留完整 pyodide
- 云端部署：使用 CDN

### Q2: 自定义域名怎么配置？

**Vercel:**
1. Settings → Domains
2. 添加你的域名
3. 按提示配置 DNS

**GitHub Pages:**
1. 仓库根目录创建 `CNAME` 文件
2. 内容：`yourdomain.com`
3. DNS 配置 CNAME 记录

### Q3: API Key 安全吗？

✅ 安全！因为：
- API Key 保存在用户浏览器本地
- 不会上传到服务器
- 每次请求直接从浏览器发送到 DeepSeek
- 你的服务器只是静态文件托管

### Q4: 需要后端服务器吗？

❌ 不需要！

简牍是纯静态网站：
- HTML/CSS/JS 前端
- Pyodide 浏览器端 Python
- DeepSeek API 云端调用

无需后端服务器！
